## 📌 Overview

DataLens AI is a professional no-code data analytics platform that automatically performs exploratory data analysis, statistical profiling, visualization, correlation intelligence, trend detection, and business insight generation on user-uploaded datasets.

Users can upload any CSV dataset and instantly receive interactive visualizations, statistical summaries, data quality assessments, and automated analytical insights without writing code.

## ⚙️ Local Installation

Clone the repository:

```bash
git clone https://github.com/SHREYA-79/datalens-ai.git
```

Navigate to project directory:

```bash
cd datalens-ai
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run application:

```bash
streamlit run app.py
```

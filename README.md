# AI Data Analyzer

AI Data Analyzer is an interactive web application that allows users to upload CSV datasets and instantly generate statistical summaries, visualizations, and AI-powered insights.

The app performs automatic exploratory data analysis and helps users quickly understand patterns, correlations, and anomalies in their data.

Features

- Upload and analyze CSV datasets
- Dataset preview and statistical summary
- Interactive data visualizations
- Correlation heatmap
- AI-generated insights using LLM APIs
- Simple and interactive Streamlit interface

Technologies Used

- Python
- Streamlit
- Pandas
- Matplotlib
- Seaborn
- Groq LLM API
